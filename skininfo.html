Berona Theme for B2Evolution 3.x

Design and Skin by Mheadley

This skin is an original skin designed by Mheadley and is meant for use in B2 3.x. This skin does not play nice with the gallery plugin and/or the lightbox plugin. IF you use either plugin, please note that you must remove the lines from the plugin that call for the Jquery files. This skin comes with updated versions of those files and you cannot run two versions of Jquery library on the same page. Thank you.
